Display
-------
property specifies how an element is displayed.

1) inline -> do not start on a new line, width is limited to what is needed. (span,a,img)
2) block  -> starts on a new line, takes up the full width available. (h1,div,p,form,header,footer)
3) inline-block  -> upto the content it is inline, for empty space it is block.

1) display: block;
2) display: inline;
3) display: inline-block;
4) display: none;

1) visibility: hidden;  //space is left empty
   display: none;       //space is given to another ele
